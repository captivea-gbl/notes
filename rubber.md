
## Using rubber

# Rubber home

https://github.com/wr0ngway/rubber

# AWS credentials

Put your AWS credentials here in a `~/.ec2/rubber-secret.yml` file: 

```
cloud_providers: 
  aws: 
    access_key: XXX 
    secret_access_key: YYY 
    account: ZZZ 
```

# Install rubber gem

```
gem install rubber
```
